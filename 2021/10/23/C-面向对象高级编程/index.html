参数返回 return by value or reference(to const)

尽量 return by reference

什么时候不能返回引用：如果函数内部新创建的变量要返回，就不能返回引用，应该返回指针，否则随着函数结束变量就会消亡

如果得到的变量只是在原有空间上修改就可以返回引用
