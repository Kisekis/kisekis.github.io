仿射无关与线性无关

从直观上讲，线性无关Linearly Independent就是最少可以Span成一个n维空间的向量，数量为n个

同理，仿射无关Affinely Independent就是最少可以Generate成n维仿射包Affine hull的向量，数量为n个

仿射无关 Affinely Independent

定义

仿射包Affine hull：

C为集合

称为C的仿射包

仿射包的意义为包含C的最小仿射集

仿射集：

意义为集合C中连接任意两点构成的直线都在集合C中，称C为仿射集Affine set

线性方程的的解是仿射集，任意一个仿射集都可以表示为某个线性方程的解集

联想：凸集Convex set

和仿射集的区别就是属于

换句话讲就是任意两点连接成的线段都在C内

仿射维度Affine dimension：

C的仿射维度即为C的仿射包的维度

个点生成 维的仿射包

Simplexes 单纯形

注意这有个点，这个点是Affinely Independent的，也就是说可以generate一个维的仿射包

另外不是仿射集！！！可以generate仿射包

举例

  • 一维单纯形(2个点) ： 线段

  • 二维单纯形(3个点) : 三角（包含内部）

  • Unit simplex 单位单纯形

    是n维向量，则单位单纯形生成的仿射包为n维

  • Probability simplex :

    是n维向量，则单位单纯形生成的仿射包为n-1维
