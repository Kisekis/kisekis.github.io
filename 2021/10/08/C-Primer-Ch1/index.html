输入输出

输入输出库

iostream 库，包含 istream 输入流和 ostream 输出流。

cin是istream的一个对象

cout(标准输出)，cerr(错误信息)，clog(一般性信息)是ostream的对象

endl 可以结束当前行,并将缓冲区(buffer)中的内容刷到设备中

要及时刷新缓冲区，防止调试时有待输出信息停留在buffer中

读取数量不定的输入数据

当遇到EOF，或者无效输入(如类型不匹配)，istream对象条件会变为false。

  • Windows : Ctrl + Z -> Enter
  • Unix : Ctrl + D

  • argument 实参 Arguments are the data you pass into the method’s parameters.
