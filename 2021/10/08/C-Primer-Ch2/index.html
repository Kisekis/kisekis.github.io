int a = {0};

int a{0};

int a(0);

int a{0} 与 int a(0) 的区别在于，花括号会进行丢失信息检验，可能丢失信息则报错

e.g. long double b = 3.1415936;

​ int a{b}; //报错

​ int a(b); //不报错，但丢失信息

指针

空指针最好使用 nullptr 来定义

why different?

因为C++中不能将void 类型的指针*隐式转换成其他指针类型

而整型字面量0可以隐式转换成任意类型的空指针常量

而 nullptr 并非整型类别，甚至也不是指针类型，但是==能转换成任意指针类型==。nullptr的实际类型是std:nullptr_t。

建议初始化所有的指针，要么定义了对象后再定义指针指向

要么置为 nullptr

void* 可以存放任意类型的指针，即单纯指向一个地址的指针（不知道类型）
